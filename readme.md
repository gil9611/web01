<h3>개요.</h3>
<p>프로젝트의 설명</p>
<p>주요기능 : 웹 사이트</p>
<p>-회원관리 : 회원약관, 회원가입, 아이디 중복체크, 로그인, 로그아웃, 마이페이지, 정보수정, 회원탈퇴</p>
<p>-게시판기능(공지사항) : 글 목록, admin글 작성, admin글 수정, admin글 삭제. 회원 글읽기.</p>
<p>개발도구 : eclipse, visual studio code</p>
<p>개발언어 : java 1.8, jsp web 4.0</p>
<p>개발모델 : mvc1</p>
<p>빌드 도구 : maven</p>
<p>사용 라이브러리</p>
<p>RDBMS : mariaDB</p>
